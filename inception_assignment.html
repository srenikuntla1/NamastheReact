What is Emmet?

Ans: Emmet is a development tool which takes in abbreviations and make it full fledged code. typically works with HTML, CSS.

Difference between a Library and Framework?

Ans:

  • Library: minimum effort needs to include it in your program or application.
  • Framework: Framework is used to develop full fledged application.

What is CDN? Why do we use it?

Ans: CDN stands for Content Delivery Network.

  • CDN is used to deliver the content to the user in a faster way.
  • It is used to reduce the load on the server.
  • It is used to reduce the latency.

Why is React known as React?

Ans: React is known as React because it reacts to the changes in the state of the application. As it maintains virtual dom before it renders to the screen.

What is crossorigin in script tag?

Ans: crossorigin is an attribute in the script tag which is used to load the script from the different domain. It is used to load the script from the different domain.

What is the difference between React and ReactDOM?

Ans:

  • React: React is a library which is used to create the components.
  • ReactDOM: ReactDOM is a library which is used to render the components to the screen.

What is the difference between react.development.js and react.production.js files via CDN?

Ans:

  • react.development.js: It is used for development purpose. It is not minified and it has some additional warnings and error messages.
  • react.production.js: It is used for production purpose. It is minified and it does not have any additional warnings and error messages.

What is async and defer?

Ans:

  • async: It is used to load the script asynchronously. It does not block the rendering of the page.
  • defer: It is used to load the script after the rendering of the page. It does not block the rendering of the page.
